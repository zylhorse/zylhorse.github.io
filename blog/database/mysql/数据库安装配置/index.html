zylhorse

5分钟

ubuntu 安装

sudo rm /var/lib/mysql/ -R
sudo rm /etc/mysql/ -R
sudo apt-get autoremove mysql-server mysql-common --purge
sudo apt-get remove apparmor // 输入Y回车
sudo apt-get install mysql-server mysql-common //重新安装

windows 安装

  1. 下载地址
  2. 解压缩配置my.ini
[mysql]
# 设置mysql客户端默认字符集
default-character-set=utf8
 
[mysqld]
# 设置3306端口
port = 3306
# 设置mysql的安装目录
basedir=G:\mysql-8.0.15-winx64
# 设置mysql的data目录
datadir=G:\mysql-8.0.15-winx64\data

# 允许最大连接数
max_connections=20
# 服务端使用的字符集默认为8比特编码的latin1字符集
character-set-server=utf8
# 创建新表时将使用的默认存储引擎
default-storage-engine=INNODB
  1. 安装
# 用管理员权限启动命令行，最后会提示root密码
mysqld --initialize --console
  1. 安装mysql服务
mysqld install
  1. 启动mysql服务
net start mysql

mysql 配置

修改原因：

mysql8 之前的版本中加密规则是mysql_native_password,而在mysql8之后,加密规则是caching_sha2_password 把mysql用户登录密码加密规则还原成mysql_native_password.

两种方法

ALTER USER ‘root’@’localhost’ IDENTIFIED BY ‘password’ PASSWORD EXPIRE NEVER; #修改加密规则 
flush privileges;

ALTER USER ‘root’@’localhost’ IDENTIFIED WITH mysql_native_password BY ‘password’; 
flush privileges;

远程访问配置

  1. 修改/etc/mysql/mysql.conf.d/mysqld.conf
bind-address 127.0.0.1
=> 
bind-address 0.0.0.0
  1. 创建远程访问用户,不添加域名
create user "username" identified by "password"
grant all privileges on *.* to "username"@"%" with grant option;
flush privileges;
  1. 修改用户远程访问
use mysql
update user set host="%" where User=username;
grant all privileges on *.* to "username"@"%" with grant option;
flush privileges;
  1. 回收用户权限
revoke all privileges on *.* from "username"@"%";
  1. 查看权限
show grants for "username"@"%";

修改mysql 最大连接数 无限制

修改linux mysql 用户打开文件句柄数

sudo vim /etc/security/limits.conf 

mysql 		 soft 	 nofile 	 65535
mysql 		 hard 	 nofile		 65535

修改mysql 文件句柄限制

sudo vim /lib/systemd/system/mysql.service 

LimitNOFILE=infinity
LimitMEMLOCK=infinity

修改mysql 最大连接数

sudo vim /etc/mysql/mysql.conf.d/mysqld.cnf

max_connections=5000
wait_timeout=600
interactive_timeout=600

docker 修改ulimit

ulimits:
                nofile:
                        soft: "65536"
                        hard: "65536"
