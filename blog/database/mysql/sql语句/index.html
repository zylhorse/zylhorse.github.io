MySQL常用sql语句

2014-11-20

zylhorse

5分钟

创建数据库

mysql> create database "dbname";

创建用户

mysql> create user "username"@"host" identified by "password";

赋予用户权限

mysql> grant all privileges on dbname.* to "dbuser"@"host" with grant option;

查看权限

mysql> show grants for "username"@"host";

删除权限

mysql> revoke all on dbname.table from 'user'@'%';

备份还原

-d 备份表结构
-t 备份数据

备份数据库

mysqldump -hys1.51asj.com -P3306 -uroot -p --databases dbname > /tmp/dbname.sql

备份数据库单表

mysqldump -hys1.51asj.com -P3306 -uroot -p dbname tablename > /tmp/tablename.sql

还原数据

mysql> source /tmp/dbname.sql
