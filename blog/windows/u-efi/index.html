Windows系统删除U盘EFI分区

2016-09-05

zylhorse

5分钟

  1. 管理员权限启动CMD
  2. 输入磁盘分区命令： diskpart 
    C:\Users\Administrator>diskpart   
   
DISKPART>
  3. 查看所有磁盘： list disk 
    DISKPART> list disk
    
   磁盘 ###  状态           大小     可用     Dyn  Gpt
   --------  -------------  -------  -------  ---  ---
   磁盘 0    联机              111 GB      0 B
   磁盘 1    联机              465 GB      0 B
   磁盘 2    联机              111 GB      0 B
   磁盘 3    联机             7648 MB   809 MB
  4. 选择要进行操作的磁盘： select disk 3 
    DISKPART> select disk 3
   
磁盘 3 现在是所选磁盘。
  5. 清理磁盘： clean 
    DISKPART> clean
   
DiskPart 成功地清除了磁盘。
  6. 重新挂载U盘： 进入计算机管理 -> 磁盘管理 -> 找到相应U盘，右键新加卷
