从每层协议及物理设备了解网络模型

2013-12-16

zylhorse

5分钟

OSI七层模型

  1. 物理层Physical： IEEE 802.1A、IEEE 802.2、IEEE 802.11
  2. 物理链路层Data Link: FDDI、Ethernet、Arpanet、PDN、SLP、PPP
  3. 网络层Network：IP、ICMP、ARP、RARP、AKP、UUCP
  4. 传输层Transport：TCP、UDP
  5. 会话层Session：SMTP、DNS
  6. 表示层Presentation：Telnet、rlogin、SNMP、Gopher
  7. 应用层Application：TFTP、FTP、NFS、WAIS、HTTP

TCP/IP 五层模型

  1. 物理层：中继器，集线器
  2. 数据链路层： 二层交换机
  3. 网络层： 路由器，三层交换机
  4. 传输层： 四层交换机
  5. 应用层
