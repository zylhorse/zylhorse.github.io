Hugo评论系统-Utteranc

2020-03-07

zylhorse

5分钟

  • 安装Utteranc

    • 进入Utteranc GitHub App
    • 在页面中选择你要添加的repo,进行安装

  • 配置config.toml

    ## Utterances
[params.utterances]
    enable = true
    repo = "zylhorse/zylhorse.github.io" // 这里替换为你的github repo
    issueTerm = "title" // issue 标题， 基于当前文章标题/路径..
    theme = "github-light" // 风格
    themeDark = "photon-dark"

    参考： https://utteranc.es/

  • 配置comments.html 在layouts/_default/comments.html中添加一下代码：

    {{ if .Site.Params.utterances.enable }}
  <label>{{ jsonify .Params.categories }}</label>
  <article class="ucomments">
    <script src="https://utteranc.es/client.js"
            repo={{ .Site.Params.utterances.repo }}
            issue-term={{ .Site.Params.utterances.issueTerm }}
            label={{ jsonify .Params.categories }} // 以categories作为issue的labels
            theme={{ .Site.Params.utterances.theme }}
            crossorigin="anonymous"
            async>
    </script>
  </article>
{{ end }}

  • 配置single.html 在layouts/_default/comments.html中添加一下代码：

    {{ if not ( eq .Params.comments false) }}
       {{ .Render "comments" }}
{{ end }}
