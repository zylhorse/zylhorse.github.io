Ubuntu安装配置无线网卡

2014-06-11

zylhorse

5分钟

查看所有网络

$ ifconfig -a

打开无线网卡

$ ifconfig wlan0 up

wlan 工具

$ apt-get install wpasupplicant

连接无线网络

wpa_passphrase 无线网络SSID无线网络密码 > 配置文件名

配置无线网络

  1. $ auto wlan0
  2. $ iface wlan0 inet dhcp
  3. $ wpa-conf /etc/wpa_config.conf

修改网卡名称

  1. $ vim /etc/default/grub
   
     GRUB_CMDLINE_LINUX="net.ifnames=0 biosdevname=0"
  2. $ update-grub

若提示没有此命令，请先输入安装命令

$ apt-get install grub2-common
