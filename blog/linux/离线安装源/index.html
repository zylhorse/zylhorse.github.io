Ubuntu依赖包的离线打包和安装

2015-02-02

zylhorse

5分钟

Ubuntu 16.04离线安装步骤

宿主机打包

  1. 清理apt 缓冲区$ sudo rm -rf /var/cache/apt/archives/*
  2. 下载所需组件$ sudo apt-get -d install <包名>
  3. 创建目录，并拷贝 
    $ mkdir <your-path>
$ cp -r /var/cache/apt/archives <your-path>

修改名目权限

$ chown -R <your-path>

建立依赖关系

$ sudo apt install dpkg-dev
$ cd <your-path>
$ sudo dpkg-scanpackages archives /dev/null  | gzip > <your-path>/archives/Packages.gz

拷贝目录到离线系统

$ scp -r <your-path> <离线系统>

离线系统步骤

修改apt源

$ sudo vi /etc/apt/sources.list
清空并添加
deb file:///<your-path> archives/

安装指定组件

$ sudo apt-get update 
$ sudo apt-get install <包名>
